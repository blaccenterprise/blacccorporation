What Is HTML?

HTML stands for Hypertext Markup Language. HTML is one of many aspects that are used in coding, but the main purpose of HTML is to give your program the skeletal structure on the front-end when executed in a browser. For example, the code from New.html:

<!DOCTYPE>

<html>

<head>

<style>

body {

background-color: black;

color: gold;

font-family: Times;

}

h1  {

color: gold;

font-family: Times;

font-size: 65px;

}

</style>

</head>

<body>

<h1>

More Life

</h1>

</body>

</html>

will display a black screen with the two words “More Life” in gold. The word “More Life” is the HTML in practice.
